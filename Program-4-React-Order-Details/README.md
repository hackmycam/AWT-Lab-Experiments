# Experiment 5: React State Management

Develop a React application demonstrating state management with hooks.

## Requirements
- Node.js (v18+)
- npm

## Setup & Install
```bash
npm install
```

## Dependencies
- react
- react-dom
- vite (dev)

## How to Run
```bash
npm run dev
```
Then open `http://localhost:5173` in your browser.
