# Experiment 10: Person Details App

Create a NodeJS application to insert and display details of a person using HTML, Express framework, and MongoDB.

## Requirements
- Node.js (v18+)
- npm
- MongoDB (running on port 27017)

## Setup & Install
```bash
npm install
```

## Dependencies
- express
- mongoose

## How to Run

1. **Start MongoDB**:
   ```bash
   mongod
   ```

2. **Start Server**:
   ```bash
   node server.js
   ```

## Usage

1. Open `http://localhost:3000` in your browser.
2. Enter a Name and Age, then click "Save".
3. To view all saved data, navigate to `http://localhost:3000/show`.
