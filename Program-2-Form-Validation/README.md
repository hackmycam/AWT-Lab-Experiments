# Experiment 2: Form Validation Using JavaScript

Validate a form using JavaScript with fields for name, email, phone, and password.

## Requirements
- A modern web browser (Chrome, Firefox, Edge)
- No additional installations needed

## Files
- `form.html` - HTML form with JavaScript validation

## How to Run
Simply open `form.html` in your browser:
```
Right-click form.html → Open with → Browser
```
Or use VS Code Live Server extension.
