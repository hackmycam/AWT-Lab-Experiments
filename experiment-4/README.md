# Experiment 4: React Component Lifecycle Methods

Understand and implement React component lifecycle methods (componentDidMount, componentDidUpdate, componentWillUnmount).

## Requirements
- Node.js (v18+)
- npm

## Setup & Install
```bash
npm install
```

## Dependencies
- react
- react-dom
- vite (dev)

## How to Run
```bash
npm run dev
```
Then open `http://localhost:5173` in your browser.
