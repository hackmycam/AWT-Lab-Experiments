# Experiment 3: React Components - Employee & Events Demo

Develop a React application with multiple components demonstrating employee data and event handling.

## Requirements
- Node.js (v18+)
- npm

## Setup & Install
```bash
npm install
```

## Dependencies
- react
- react-dom
- vite (dev)

## How to Run
```bash
npm run dev
```
Then open `http://localhost:5173` in your browser.
