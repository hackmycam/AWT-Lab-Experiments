# Experiment 6: React Router Navigation

Develop a React application demonstrating client-side routing with React Router.

## Requirements
- Node.js (v18+)
- npm

## Setup & Install
```bash
npm install
```

## Dependencies
- react
- react-dom
- react-router-dom
- vite (dev)

## How to Run
```bash
npm run dev
```
Then open `http://localhost:5173` in your browser.
